# **PWD-MGR**
Um gerenciador de senhas _**simples**_, feito em C# para um projeto escolar.

## ✨ Funções
* GUI em Console
* Criptografia de senhas
* Criador de senhas aleatórias
* Conselheiro de segurança
* Código aberto!

### 🛠️ Feito Com
.Net Framework 6.0 e Visual Studio Community 2022.
